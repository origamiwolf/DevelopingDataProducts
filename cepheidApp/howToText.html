How to Use

Introduction Tab: Introduction and Background for this Shiny app.

How to Use Tab: This tab.

Data Tab: This tab shows the data being used for the Period-Luminosity model.

Plot Tab: This tab shows plots of the data. The appearance of the plot is controlled by the following widgets:

  • Distance to SMC: This sets the distance to the Small Magellanic Cloud. The computation of the absolute magnitudes is dependent on this variable. Initial value of 200 kilo light years is obtained from Reference 5.
  • Axis Type: Select whether to display x-axis as Period or log10Period.
  • Display magnitude type: Select whether to display apparent magnitude or absolute magnitude on y-axis.
  • Display regression lines: Select whether to display model fits of magnitude vs log10Period.

Distance Estimator Tab: In this tab, the model fits shown in the Data tab can be used to estimate the distance to other Cepheids whose period and apparent magnitudes are known. To perform an estimate, simply input the period of the Cepheid (in days), and either one or both of its apparent magnitude extremes. From these values, predictions of its absolute magnitudes and distance will be made.
